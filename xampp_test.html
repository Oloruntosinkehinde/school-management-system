üîß Common Issues & Solutions

Issue 1: "Page not loading" or "404 Error"

Solution: Make sure you're accessing http://localhost/school_management/ not localhost:8000

Issue 2: "Database connection failed"

Solution:

  1. Open phpMyAdmin
  2. Create database named school_management
  3. Import the file backend/sql/comprehensive_schema.sql

Issue 3: "Buttons not working" or "JavaScript errors"

Solution:

  1. Press F12 to open Developer Tools
  2. Check Console tab for JavaScript errors
  3. Check Network tab to see if CSS/JS files are loading

Issue 4: "CSS not loading" or "Page looks unstyled"

Solution: Check that css/main.css and css/tailwind.css exist and are readable
